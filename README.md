# RFM 會員招式訊息生成器

這是一個簡單的 RFM 行銷文案生成工具。

## 功能

- 選擇會員分群（RFM）
- 自動推薦行銷招式
- 自動生成 5 則訊息內容
- 支援手動調整招式
- 每次生成都不同（動態隨機）

## 使用方式

1. 下載專案
2. 開啟 `index.html`
3. 選擇分群
4. 點擊「產出內容」

## 部署到 GitHub Pages

1. 上傳到 GitHub Repo
2. 到 Settings → Pages
3. Source 選擇：
   - Branch: `main`
   - Folder: `/root`
4. 開啟網址即可使用

## 技術

- Vue 3 (CDN)
- 純前端（無後端）
